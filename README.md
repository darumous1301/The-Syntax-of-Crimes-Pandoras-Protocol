# The-Syntax-of-Crimes-Pandoras-Protocol
A Java console-based detective game where Sherlock Holmes is on vacation, and the user takes his place to solve crimes. Holmes gives cryptic hints online to outsmart Moriarty. Each solved case earns clues leading to the ultimate mystery—Pandora’s Box.
